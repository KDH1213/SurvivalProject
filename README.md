# 프로젝트 소개
신비한 섬에 표류한 당신.
섬의 중심, 붕괴되고 있는 코어를 점령하고 지켜내야 살아남을 수 있습니다.
사나워진 야생 동물들의 끊임없는 공격 속에서 생존, 건설, 전투, 전략을 모두 활용해
섬의 비밀을 파헤쳐 보세요!

### 장르
롤플레잉, 어드벤쳐
### 개발 도구
